Discuz数学公式插件，用于在Discuz论坛显示LaTeX数学公式。使用MathJax渲染。

## 插件信息

| -------- | -------- |
| 插件名称 | LaTeX数学公式 |
| 插件ID | zxsq_mathjax |
| 插件功能 | 显示LaTeX数学公式 |
| 插件URL | <http://addon.discuz.com/?@zxsq_mathjax.plugin> |
| 支持网站 | <http://mathjax.tecbbs.com> |

## 使用方法

```
1. [math][/math]标签包裹公式，跨行公式加 \\
2. $ $ 或者\(\)包裹行内公式
3. $$ $$或者\[\]包裹跨行公式
```
